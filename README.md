# Hotel Reservation System

A Java-based Hotel Reservation System developed as a university project. This repository currently contains the **Milestone 1** implementation — the backend classes with full object-oriented design.


# Project Description

The Hotel Reservation System allows guests to register, browse available rooms, make reservations, and handle payments. Staff members (Admins and Receptionists) can manage rooms, reservations, and guest information through the system.

# Features (Milestone 1)

-  **Guest Management** — register, login, and manage guest profiles
-  **Room Management** — different room types with pricing and amenities
-  **Reservation System** — create, view, and cancel reservations
-  **Payment Processing** — multiple payment methods with invoice generation
-  **Staff Management** — admin and receptionist roles with different permissions
-  **Custom Exceptions** — proper error handling for invalid operations
-  **Interfaces & Abstract Classes** — clean OOP design with `Payable`, `Manageable`, and `AdminReceptionist`
